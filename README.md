# 前提
* 安装nodejs必须
* 安装node-dev(非必需)  


#运行
1. 下载该文件夹下所有文件  
2. 在根目录下运行node index.js 或者node-dev index.js   
3. 打开浏览器[http://localhost:8888/](http://localhost:8888/)就可以查看到效果了


#备注
* index.js主要的入口文件
* server.js服务器模块
* router.js路由模块
* requestHandlers.js请求处理模块
